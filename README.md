# Thermal Camera’s Potential in Active Safety

## Project Overview
This student project investigates the **potential of thermal vision cameras for enhancing active vehicle safety functions**.  
The focus is on exploring how **temperature-based information** can provide additional insights about detected objects — leading to **improved classification and risk assessment**.

The project also evaluates the **performance of thermal imaging under reduced visibility conditions** (e.g., fog, night driving) to assess its advantages over conventional sensors.  
By combining **sensor data analysis** with **practical testing**, the project contributes to the development of **advanced driver assistance systems (ADAS)**.

---

## Objectives
- Evaluate how thermal imaging improves object detection and classification in ADAS.  
- Compare thermal sensors with conventional visible-light cameras under poor visibility conditions.  
- Analyze thermal image data using Python-based algorithms for object recognition and pattern analysis.  
- Assess the integration potential of thermal cameras into vehicle safety systems.
